## Setup Instructions

1. Clone the repo  
   `git clone https://github.com/cherrycatzap/library_manager.git`

2. Navigate to the directory  
   `cd library_manager`

3. Install dependencies  
   `pip install -r requirements.txt`

4. Run the server  
   `python manage.py runserver`
